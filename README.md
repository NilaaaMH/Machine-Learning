**Mini Project - Machine Learning: Klasifikasi Kategori Pencurian Berdasarkan Teks Putusan Pengadilan**

Deskripsi
Proyek ini mengimplementasikan algoritma machine learning untuk mengklasifikasikan kategori tindak pidana pencurian (pencurian biasa vs. pencurian berat) berdasarkan analisis teks putusan Pengadilan Negeri Samarinda tahun 2024â€“2025. Dataset diperoleh melalui scraping dari situs Mahkamah Agung dan diproses melalui tahap preprocessing, ekstraksi fitur, dan klasifikasi menggunakan model seperti Naive Bayes, SVM, Random Forest, LSTM, dan IndoBERT (transfer learning).
Tujuan utama adalah meningkatkan efisiensi analisis hukum dengan otomatisasi klasifikasi, membantu lembaga peradilan dalam mengelola data putusan.
